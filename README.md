# Flipkart Sales Dashboard

## Overview

This project is an interactive Excel dashboard created to analyze Flipkart sales data and provide insights into business performance. The dashboard helps track sales trends, product performance, revenue, and customer-related metrics through visual reports.

## Objectives

* Analyze overall sales performance.
* Identify top-selling products and categories.
* Monitor revenue trends.
* Track customer purchasing patterns.
* Create an interactive and user-friendly dashboard.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning and Transformation

## Dashboard Features

* Total Sales Analysis
* Revenue Tracking
* Category-wise Performance
* Product-wise Sales Analysis
* Interactive Filters and Slicers
* Visual Reports and Charts

## Key Insights

* Identified top-performing product categories.
* Analyzed monthly sales trends.
* Evaluated revenue contribution by products.
* Improved data visualization for better decision-making.
